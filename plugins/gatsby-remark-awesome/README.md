# gatsby-remark-awesome

This plugin helps convert `awesome-X` markdown README.md's into LekoArt's Status Dashboard template.

Example:
[Awesome BDD](https://github.com/omergulen/awesome-bdd) into [BDD Tools](https://bdd.tools).
